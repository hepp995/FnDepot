<p><strong>EasyTier是一个由 Rust 和 Tokio 驱动的简单、安全、去中心化的异地组网方案。</strong></p><p><strong>本项目为飞牛商店EasyTier-Web的修改版，添加自定义用户名，适配x86、arm64，支持双模式启动。</strong></p><p><strong><br>使用方法：</strong></p><p>web控制台模式(推荐)：API主机填飞牛本机ip+默认11211端口或者反代后的地址。用安装向导中填的用户名进行注册，登陆后在设备列表添加一个新的网络。</p><p>配置文件模式：在应用文件/EasyTier/config下修改或添加toml文件，重启生效。</p><p>详情可以看easytier官网介绍。如果想要自定义web端口，请在应用设置修改，重启生效。</p><p>可在web控制台设置好后导出toml配置，再以配置模式运行。</p>
